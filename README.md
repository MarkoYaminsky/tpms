## Лабораторна 2
Виконайте інструкції до завдань А та Б
   1) Замініть дані з файлу ***labs/lab_2/task1*** на свої за таким принципом:
      - В першому рядку є чотири числа (порядок важливий): 
      ліва межа першого інтервалу, довжина інтервалу, та значення функцій t(γ), q(γ)
      для ваших параметрів (шукаєте на с. 73) 
      - В другому рядку є сім чисел: це кількість випадкових подій у відповідному інтервалі
   2) Замініть дані з файлу ***labs/lab_2/task2*** на свої за таким принципом:
      - Перше число - це повна кількість експериментів
      - Друге число - це кількість успішних експериментів 
      - Третє число - це таке значення, для якого 
        Ф(х) в табл. 3 (с. 71-72) рівне γ/2 з вашого варіанту
      
Після цього запустіть ***labs/lab_2/results.py***
